HTML辞典

タグ 読み 説明 代表的な属性
<p> ピー 段落（文章のまとまり）を表す class, id
<a> エー リンクを作成する href, target
<img> イメージ 画像を表示する src, alt, width, height
<div> ディブ ブロック要素をまとめる class, id
<header> ヘッダー ページやセクションの先頭部分 class, id
<main> メイン ページの主要コンテンツ class, id
<footer> フッター ページの末尾部分 class, id
<section> セクション 章やまとまりを表す class, id
<article> アーティクル 独立した記事やコンテンツ class, id
<nav> ナブ ナビゲーション領域 class, id
<aside> アサイド 補足情報やサイドバー class, id
<span> スパン インライン要素をまとめる class, id
<strong> ストロング 重要な強調（太字） class
<br> ビーアール 改行を挿入する なし
<em> エム 弱い強調（斜体） class
<mark> マーク ハイライト表示 class
<code> コード コードを表す class
<pre> プレ 整形済みテキスト class
<video> ビデオ 動画を表示する src, controls
<audio> オーディオ 音声を再生する src, controls
<source> ソース 動画や音声のソース指定 src, type
<figure> フィギュア 図や画像をまとめる class
<figcaption> フィグキャプション 図の説明文 class
<ul> ユーエル 箇条書きリスト class
<ol> オーエル 番号付きリスト class
<li> エルアイ リスト項目 なし
<table> テーブル 表を作成する class
<tr> ティーアール 表の行 なし
<th> ティーエイチ 見出しセル scope
<td> ティーディー データセル colspan, rowspan
<input> インプット フォーム入力欄 type, name, value
<button> ボタン ボタンを作成する type, class
<form> フォーム フォーム全体 action, method
<label> ラベル 入力欄の説明 for
<textarea> テキストエリア 複数行の入力欄 name, rows, cols
<select> セレクト プルダウンメニュー name
<option> オプション 選択肢 value
<fieldset> フィールドセット フォーム項目のグループ化 class
<legend> レジェンド グループのタイトル class
<html> エイチティーエムエル HTML文書のルート要素。lang属性で文書の言語を指定 lang
<head> ヘッド メタ情報・設定・外部リソースの読み込みをまとめる なし
<meta> メタ 文字コード・SEO・表示設定などのメタ情報を指定 charset, name, content
<title> タイトル ブラウザのタブに表示されるページタイトル なし
<link> リンク 外部CSSなどのリソースを読み込む rel, href
<script> スクリプト JavaScriptを読み込む。外部ファイルはsrc属性 src, defer
<base> ベース 相対URLの基準となるURLを指定する href, target
<style> スタイル CSSをHTML内に直接記述する type
<noscript> ノースクリプト JavaScriptが無効な環境で代替内容を表示する なし
