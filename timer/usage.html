使用方法

このアプリは、複数のカウントダウンタイマーを同時に管理できるツールです。

基本操作

  • ＋ 新しいタイマー：タイマーを追加します。
  • 時・分・秒・ms：カウントダウン時間を入力します。
  • Start：カウントダウンを開始します。
  • Stop：カウントダウンを停止します（アラーム音も停止）。
  • Reset：残り時間を 0 にクリアし、アラーム音も停止します。
  • ×：タイマーを削除します（アラーム音も停止）。

アラーム音について

タイマーが 0 になるとアラーム音が鳴ります。 Stop / Reset / 削除 のいずれでもアラーム音は自動的に停止します。
